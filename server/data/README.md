# data

JSON data files live here — the static data that drives the app.

Likely future contents:
- Parts catalog (CPUs, motherboards, RAM, etc.)
- Step-block content for the build guide
- Validation rule definitions

Nothing here yet. The shape of this data hasn't been designed — that
happens when the parts-selection / build-guide features get built.

## For teammates contributing data

You can contribute here without writing code — adding parts, writing
step content. Talk to the team about the current format before adding
files, since it's still being figured out.
